Aplikasi Mobile Kantin Pintar Kampus

Proyek penyelamatan aplikasi kantin kampus menggunakan Scrum, fokus pada login, pilih menu, dan pembayaran.

Lihat Fitur

Tentang Proyek

Proyek ini sebelumnya mandek di 50% karena kode berantakan, dokumentasi API tidak ada, serta konflik komunikasi antara backend dan frontend. Tim baru membuat rencana kerja 2 minggu untuk memperbaiki proses kerja dan menyelesaikan fitur utama aplikasi.

Fitur Utama

Login

Mahasiswa dapat masuk ke aplikasi dengan validasi token dan pengujian skenario login.

Pilih Menu

Daftar menu kantin tampil rapi dengan kategori dan format data JSON yang distandarkan.

Pembayaran

Pengguna dapat melakukan checkout dan pembayaran mode sandbox sebelum aplikasi live.
